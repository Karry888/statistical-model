## 一、项目总览
### 1.1 核心研究背景
在我国“双碳”目标持续推进、欧盟CBAM碳边境调节机制落地的背景下，广东省作为全国第一大铝消费省份、第一大铝加工品出口省份，对上游电解铝的低碳属性提出了系统性强制要求，倒逼电解铝产能向云南、四川、内蒙古等绿电资源富集区转移，形成了显著的产业空间重构趋势。

### 1.2 核心研究逻辑（唯一主线，全员必须严格遵守）
```
核心因果链：广东消费端低碳约束（驱动因）→ 云南/四川/内蒙古电解铝产能空间重构（研究果）
├─ 驱动端：广东低碳采购需求、CBAM出口倒逼、省级碳足迹管控政策
├─ 研究对象：云/川/蒙3省电解铝产能扩张与空间集聚
└─ 研究目标：验证因果关系、量化倒逼效应、识别异质性特征、模拟政策冲击
```

### 1.3 核心研究问题
1.  广东消费端低碳约束是否对云/川/蒙电解铝产能空间重构存在显著的倒逼效应？
2.  该倒逼效应的大小、非线性特征与异质性规律是什么？
3.  政策冲击下，不同绿电模式的产能响应有何差异？

## 四、模型建立全方案（按顺序执行，可复现、可验证）
### 4.1 模型执行优先级
```
1. 基准模型：双向固定效应面板回归 → 验证核心因果关系
2. 因果识别：DID双重差分模型 → 解决内生性问题，强化因果识别
3. 创新模型：GAM广义加性模型 → 捕捉非线性倒逼效应（竞赛核心亮点）
4. 稳健性检验：多维度验证结论可靠性
5. 拓展分析：情景模拟、异质性分析
```

### 4.2 核心模型详细设定
#### 模型1：基准模型 - 双向固定效应面板回归
**核心目的**：验证广东消费端低碳约束对云/川/蒙电解铝产能的倒逼效应是否存在
**模型公式**：
$$Capacity_{it} = \beta_0 + \beta_1 Demand_{it} + \beta_2 Export_{t} + \beta_3 Policy_{t} + \sum\gamma_k Controls_{kit} + \mu_i + \lambda_t + \varepsilon_{it}$$
**变量定义**：
| 变量符号 | 变量名称 | 对应数据清单 |
|----------|----------|--------------|
| $Capacity_{it}$ | 被解释变量：i省t年电解铝运行产能 | 第一类指标1 |
| $Demand_{it}$ | 核心解释变量1：广东从i省t年电解铝采购量 | 第二类指标1 |
| $Export_{t}$ | 核心解释变量2：广东t年对欧盟铝制品出口额 | 第二类指标2 |
| $Policy_{t}$ | 核心解释变量3：广东低碳政策冲击虚拟变量 | 第二类指标3 |
| $Controls_{kit}$ | 控制变量集合 | 第三类全部指标 |
| $\mu_i$ | 省份个体固定效应 | - |
| $\lambda_t$ | 时间固定效应 | - |
| $\varepsilon_{it}$ | 随机误差项 | - |
**核心输出**：基准回归结果表、核心系数显著性与经济含义解释

---

#### 模型2：因果识别 - DID双重差分模型
**核心目的**：解决内生性问题，验证政策冲击的净效应，做实因果关系
**模型设定**：
- 处理组：云南、四川、内蒙古（受广东低碳需求冲击的绿电产能承接区）
- 控制组：山东、河南（传统火电铝产区，不受广东低碳需求拉动）
- 政策冲击时点：2020年（双碳目标提出）
**模型公式**：
$$Capacity_{it} = \beta_0 + \beta_1 Treat_i \times Post_t + \sum\gamma_k Controls_{kit} + \mu_i + \lambda_t + \varepsilon_{it}$$
**核心输出**：DID回归结果表、平行趋势检验图、安慰剂检验结果

---

#### 模型3：创新模型 - GAM广义加性模型
**核心目的**：捕捉广东需求对产能的非线性倒逼效应，是竞赛核心创新亮点
**模型公式**：
$$Capacity_{it} = \beta_0 + f(Demand_{it}) + \beta_2 Export_{t} + \beta_3 Policy_{t} + \sum\gamma_k Controls_{kit} + \mu_i + \lambda_t + \varepsilon_{it}$$
**核心说明**：用非参数平滑函数$f(Demand_{it})$替代线性项，无需预设函数形式，自动拟合广东采购需求对产能的非线性边际效应
**实现工具**：优先R语言`mgcv`包，Python可用`pygam`包
**核心输出**：平滑函数边际效应曲线图、非线性效应显著性检验结果、拟合优度对比表

---

#### 模型4：稳健性检验方案（必做，竞赛评分红线）
| 检验类型 | 操作方法 | 对应论文章节 |
|----------|----------|--------------|
| 替换被解释变量 | 用「电解铝实物产量」替换「运行产能」重新回归 | 第5章 |
| 替换核心解释变量 | 用「广东绿电铝采购占比」替换「采购量绝对值」重新回归 | 第5章 |
| 缩尾处理 | 对连续变量进行1%和99%缩尾，排除异常值干扰 | 第5章 |
| 更换政策节点 | 用2024年广东碳足迹管理方案为冲击节点，重新做DID检验 | 第5章 |
| 安慰剂检验 | 随机分配处理组/控制组，重复回归1000次，验证结果可靠性 | 第5章 |

---

#### 模型5：情景模拟分析（CBAM政策）
**核心目的**：贴合政策背景，提升论文现实价值
**情景设定**：
1.  基准情景：欧盟ETS碳价维持80欧元/吨
2.  高碳价情景：碳价上涨至120欧元/吨
3.  低碳价情景：碳价下跌至50欧元/吨
**核心输出**：不同情景下吨铝附加碳成本对比表、倒逼效应差异模拟结果、竞争力分析图

## 七、竞赛避坑红线（一票否决项，全员必须牢记）
1.  **事实性错误红线**：绝对不能出现「广东为减碳外迁电解铝产能」的错误表述，核心逻辑必须严格锁定「广东消费端低碳约束倒逼上游产能空间重构」
2.  **数据质量红线**：所有数据必须可溯源、口径统一，禁止编造数据、隐瞒数据处理过程
3.  **模型严谨性红线**：所有模型必须可复现，必须做内生性处理与稳健性检验，禁止只报显著结果、不报不显著结果
4.  **逻辑闭环红线**：论文所有内容必须围绕核心因果链展开，禁止出现与核心主题无关的冗余内容
5.  **原创性红线**：所有内容必须原创，参考文献必须标注出处，禁止抄袭、洗稿
