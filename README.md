# 广东消费端低碳约束下电解铝产业空间重构效应研究

> 统计建模竞赛项目仓库

## 一、项目总览

### 1.1 核心研究背景
在我国"双碳"目标持续推进、欧盟CBAM碳边境调节机制落地的背景下，广东省作为全国第一大铝消费省份、第一大铝加工品出口省份，对上游电解铝的低碳属性提出了系统性强制要求，倒逼电解铝产能向云南、广西、内蒙古等绿电资源富集区转移，形成了显著的产业空间重构趋势。

### 1.2 核心研究逻辑
```
核心：广东消费端低碳约束（驱动因）→ 云南/广西/内蒙古电解铝产能空间重构（研究果）
├─ 驱动端：广东铝材出口（外部低碳需求）、广东铝材产量（本地需求代理）、低碳政策冲击
├─ 研究对象：云/桂/蒙3省电解铝产能扩张与空间重构
├─ 传导机制：产能迁移、绿电替代、工艺优化
└─ 研究目标：验证因果关系、量化倒逼效应、识别异质性特征、检验传导机制
```

### 1.3 核心研究问题
1. 广东消费端低碳约束是否对云/桂/蒙电解铝产业空间重构存在显著的倒逼效应？
2. 该倒逼效应通过"产能迁移、绿电替代、工艺优化"三个机制如何传导？
3. 政策冲击下，不同省份的产能响应有何异质性差异？

## 四、模型设定

### 4.1 模型执行顺序
```
1. 基准模型：双向固定效应面板回归 → 验证核心因果关系
2. 因果识别：DID双重差分模型 → 解决内生性，强化因果识别
3. 机制检验：中介效应/调节效应 → 检验三条传导机制
4. 创新模型：GAM广义加性模型 → 捕捉非线性倒逼效应
5. 稳健性检验：多维度验证结论可靠性
```

### 4.2 模型1：双向固定效应面板回归（基准模型）

**核心目的**：验证广东消费端低碳约束对上游省份电解铝产量的倒逼效应。

**模型公式**：
$$Output_{it} = \beta_0 + \beta_1 Export_{t} + \beta_2 GdAlOutput_{t} + \beta_3 PolicyShock_{t} + \sum\gamma_k Controls_{kit} + \mu_i + \lambda_t + \varepsilon_{it}$$

| 变量 | 说明 |
|------|------|
| $Output_{it}$ | i省t年电解铝产量（被解释变量） |
| $Export_{t}$ | 广东t年铝材出口量（核心解释变量1：外部低碳需求） |
| $GdAlOutput_{t}$ | 广东t年铝材产量（核心解释变量2：本地需求代理） |
| $PolicyShock_{t}$ | 低碳政策虚拟变量（核心解释变量3） |
| $Controls_{kit}$ | 控制变量：elec_price, grid_cef, policy_dummy, al_price |
| $\mu_i$ | 省份个体固定效应 |
| $\lambda_t$ | 时间固定效应 |

> **注意**：export、gd_al_product_output、policy_shock 为时间序列变量（仅随时间变化，不随省份变化），与时间固定效应可能存在共线性。若模型中同时纳入时间固定效应，需检验并考虑去除时间效应或改用趋势项。替代方案：export 和 gd_al_product_output 仅放入其一，或将其与省份特征交互。

### 4.3 模型2：DID双重差分模型

**核心目的**：解决内生性问题，验证政策冲击的净效应。

**模型设定**：
$$Output_{it} = \beta_0 + \beta_1 Treat_i \times Post_t + \sum\gamma_k Controls_{kit} + \mu_i + \lambda_t + \varepsilon_{it}$$

| 要素 | 设定 |
|------|------|
| 处理组 | 云南(YN)、广西(GX)、内蒙古(NMG) |
| 对照组 | 新疆(XJ)、山东(SD) |
| 政策冲击时点 | 2020年（双碳目标提出） |
| $Treat_i$ | 处理组=1，对照组=0 |
| $Post_t$ | 2020年及以后=1，否则=0 |

**关键前提**：需对处理组和对照组进行政策冲击前（2015-2019年）的**平行趋势检验**。

### 4.4 模型3：GAM广义加性模型（创新亮点）

**核心目的**：捕捉广东需求对上游产能的非线性倒逼效应。

**模型公式**：
$$Output_{it} = \beta_0 + f(Export_{t}) + \beta_2 GdAlOutput_{t} + \beta_3 PolicyShock_{t} + \sum\gamma_k Controls_{kit} + \mu_i + \lambda_t + \varepsilon_{it}$$

以非参数平滑函数 $f(Export_{t})$ 替代线性项，自动拟合出口需求对产能的非线性边际效应。

**实现工具**：R语言 `mgcv` 包（推荐），Python `pygam` 或 `statsmodels` 的 GAM 模块。

### 4.5 机制检验

| 机制 | 检验方法 | 关键变量 |
|------|----------|----------|
| 产能迁移 | 基准回归直接检验：广东需求 → 处理组产能/产量变化 | output, capacity |
| 绿电替代 | grid_cef 作为中介变量或调节变量：需求压力 → 电网清洁化 → 产能扩张 | grid_cef |
| 工艺优化 | 需额外寻找代理变量（如单位产品能耗、吨铝电耗） | 待补充 |

| 检验类型 | 操作方法 | 对应论文章节 |
|----------|----------|--------------|
| 替换被解释变量 | 用"电解铝产能"替换"产量"重新回归（产能数据作为备用） | 第5章 |

### 6.2 代码实现提示（无真实代码）

#### 数据准备阶段
1. **统一省份代码**：所有Excel文件使用一致的省份编码（建议 ISO 3166-2 或自定义编码如 YN/GX/NMG/XJ/SD），确保跨表合并时不出现匹配失败。
2. **合并为单一长面板**：将 Y、X、C 三张表按 `province + year` 键合并为一张完整的面板数据表，处理组和对照组使用相同结构。
3. **缺失值处理链**：建立缺失值处理日志，记录每个缺失值的填补方法（插值 / 趋势外推 / 行业均值），保留原始值与填补值的对照，供论文数据说明使用。
4. **变量标准化**：连续变量建议做标准化处理（z-score），使得回归系数在量纲上可比。虚拟变量不做标准化。

#### 双向固定效应（Python: `statsmodels` 或 `linearmodels`）
1. 使用 `PanelOLS` 的 `entity_effects=True` + `time_effects=True` 实现双向固定效应。
2. 注意：时间序列变量（export, gd_al_product_output, policy_shock, al_price）与时间固定效应完全共线时，模型会自动舍弃一个时间哑变量。若核心变量也因此被舍弃，需调整模型设定（如仅保留个体固定效应 + 线性时间趋势）。
3. 使用聚类稳健标准误（`cov_type='clustered'`，按省份聚类），处理面板数据潜在的组内自相关。
4. 报告 within-R²、F 统计量、各系数显著性及经济含义。

#### DID双重差分
1. 生成 `treat`（处理组=1）和 `post`（2020及以后=1）变量，以及交互项 `did = treat × post`。
2. 平行趋势检验：生成年份虚拟变量与 treat 的交互项，绘制事件研究图（event study plot）。若政策前交互项系数不显著（置信区间包含0），则平行趋势成立。
3. DID 也可以使用双向固定效应形式估计，交互项系数 $\beta_1$ 即为政策处理效应。
4. 安慰剂检验：随机打乱 treat 赋值（保持处理组数量不变），重复估计1000次，绘制 $\beta_1$ 的经验分布，真实 $\beta_1$ 应位于分布的尾部。

#### GAM广义加性模型
1. R 语言 `mgcv::gam()` 为最优选择，Python 可使用 `pyGAM` 或 `statsmodels` 的 `GLM` 结合样条基函数。
2. 关键：对核心解释变量使用平滑函数 `s(export)` 或 `s(export, by=province)`（分省份非线性曲线）。
3. 使用 `gam.check()` 诊断：检验平滑项的有效自由度（EDF），若 EDF≈1 说明关系近似线性，GAM 退化为线性模型。
4. 输出平滑函数的偏效应图（partial effect plot），展示广东出口量对产能的边际效应如何随出口量水平变化。

#### 机制检验
1. **中介效应**（绿电替代）：三步法——① 总效应：需求 → 产能；② 需求 → grid_cef；③ 需求 + grid_cef → 产能。若②中需求系数显著且③中 grid_cef 显著而需求系数下降，则中介效应存在。使用 Sobel 检验或 Bootstrap 法检验间接效应的显著性。
2. **调节效应**：在基准模型中加入 `export × grid_cef` 交互项，若交互项显著，说明电网清洁度调节了需求对产能的倒逼效应强度。

### 6.3 模型稳健性测试要点

1. **共线性诊断**：计算所有自变量的 VIF（方差膨胀因子），VIF > 10 的变量需特别关注。时间序列变量是最可能出问题的。
2. **异方差检验**：使用 Breusch-Pagan 检验判断是否存在异方差。如存在，使用稳健标准误（White/HAC）。
3. **自相关检验**：面板数据使用 Wooldridge 检验检查组内自相关。如存在，使用聚类标准误或 FGLS。
4. **单位根检验**：对每个变量做 LLC 检验或 IPS 检验，确认变量平稳。若存在单位根，考虑一阶差分后建模。
5. **模型设定检验**：使用 Ramsey RESET 检验判断是否存在遗漏的非线性项，为是否使用 GAM 提供依据。
6. **样本敏感性**：逐一剔除每个省份，重复基准回归，检验核心系数符号和显著性是否稳定。若单一省份驱动全部结果，结论不可靠。
7. **交叠DID稳健性**（如使用多期政策）：近年来 DID 文献强调交叠处理（staggered adoption）时 TWFE 估计量可能偏误，可考虑使用 Sun & Abraham (2021) 或 Callaway & Sant'Anna (2021) 的异质性处理效应估计量进行稳健性检验。

4. **逻辑闭环红线**：论文所有内容必须围绕核心因果链展开
