# statistical-model

铝产业“绿电入铝”空间重构与碳足迹空间溢出效应研究（统计建模仓库）

## 1. 项目背景与核心问题

### 1.1 现实背景

在中国“双碳”目标持续推进、欧盟 CBAM（碳边境调节机制）逐步落地、国内“能耗双控”政策持续约束高耗能行业的背景下，电解铝产业正面临“成本约束 + 碳约束 + 贸易约束”的三重压力。

### 1.2 现象观察

近年来出现明显产业迁移趋势：

- “北铝南移”
- “东铝西进”
- 产能向云南、内蒙古等绿电资源富集区集聚

### 1.3 核心研究问题

在基于能源重构的产业空间变革中：

1. 铝产业碳足迹如何在区域间重新分布？
2. 这种空间转移带来多大整体碳减排效益？
3. 是否存在可量化的空间溢出效应（排放发生地与减排受益地分离）？

### 1.4 研究意义

- 理论意义：丰富绿色产业转移与空间溢出效应的分析框架。
- 实践意义：为产业布局优化、区域协同减排、CBAM 应对策略提供量化依据。

## 2. 文献综述与理论基础（研究支柱）

本研究以三类文献为理论支撑：

1. 产业转移理论：经典产业转移理论 + 环境规制驱动产业迁移研究。
2. 碳足迹与隐含碳理论：碳足迹核算、隐含碳流、碳泄漏等概念工具。
3. 能源结构与工业减排：绿电替代对高耗能行业减排效应的实证研究。

### 2.1 本文定位与创新点

现有研究更多聚焦“单一区域减排”，对“跨区域联动与责任再分配”关注不足。本文创新点在于：

- 引入“空间溢出效应”视角，识别碳排放与减排收益的空间错配。
- 量化“绿电入铝”带来的跨区域碳减排红利。
- 将产业迁移、能源结构、国际碳壁垒（CBAM）纳入统一分析框架。

---

## 3. 理论框架与研究设计

### 3.1 核心概念界定

- 绿电入铝：依托水电、风电、光伏等可再生能源电力进行电解铝生产。
- 碳足迹：聚焦生产环节，尤其是电解环节直接与电力间接排放。
- 空间溢出效应：通过产能空间转移，将高排放区域产能迁至低排放区域，带来全国净减排并重塑区域碳责任分配。

### 3.2 机制框架：双轮驱动 - 双重红利

- 驱动轮一（推力）：高电价 + 严格能耗政策抬升传统基地成本。
- 驱动轮二（拉力）：低碳电力 + 资源禀赋 + 地方政策形成承接吸引力。
- 双重红利：
	- 经济红利：吨铝成本下降。
	- 环境红利：吨铝碳排放下降。

### 3.3 核心核算模型

减排红利核算：

$$
\Delta C = P \times E \times (EF_{old} - EF_{new})
$$

其中：

- $\Delta C$：减排量
- $P$：转移产能
- $E$：吨铝耗电
- $EF_{old}$：迁出地电网排放因子
- $EF_{new}$：迁入地电网排放因子

### 3.4 研究方法

1. 空间对比分析法：广东（消费端/高碳电网） vs 云南（绿电生产端）。
2. 情景核算法：基准情景与现状情景排放对比。
3. 隐性碳流分析：识别“排放地-收益地”分离路径。
